## Install

```shell
conan install . --build=missing
```

## Build

```shell
conan build .
```

## Features

* Image Viewer [for Image formats : [png, jpg, jpeg tiff, xpm, pcx, pnm, tiff]]
* Video Viewer [for mp4, mp3, mov]
